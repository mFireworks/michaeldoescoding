Continuous Horizon's Help Page

Our server uses a number of plugins to help enhance the survival experience. Most of these are for quality-of-life enhancements, to make life of the server easier, especially for newer players of the game. This page hopes to help you understand the systems we have in place.

- Commands -

For generic commands, most of these are self explainatory. To see which commands you have access to, type /help in the server to see a list of them. Two commands of note are /dynmap show and /dynmap hide. These commands toggle displaying your location on the dynmap. In the case you wish to hide yourself.


- Chest Protection -

To prevent people from rummaging through your stored goods, any block that has an inventory is protected by default. These blocks included, but not limited to:

  • Chests
  • Dispensers
  • Barrels
  • Woodcutters
  • Shulker Boxes
  • Etc. If you can store items in it, chances are that it's protected.
  • (Except Hoppers, Anvils, Ender Chests, Enchantment Tables, and Jukeboxes)
These blocks are protected by LWC (Lightweight Chests), which is a command-based protection plugin. By default, when any of these blocks are placed, they'll be made private where only you can access them. There's various commands you can use here to modify the protection settings of this flag, here we'll go through the following:
  • /cmodify public

    Will set a block to "public" protection, meaning anyone can access the block.

  • /cmodify private

    Will set a block to "private" protection, meaning only you and added players can access that block. (Adding players mentioned below)

  • /cmodify password (password)

    Will set a block to "password" protection, meaning you'll have to enter in the password first before you're given access.

  • /cmodify (player)

    Allows you to add or remove a player's rights to your protection. Putting a "-" infront of a player name will revoke their access.

  • /lwc flag (flag) (on/off)

    Changes the various protection flags on a block. While there are a few flags you an change, the one that are usable are hopperin, hopperout, hopper.

    • hopperin will allow items to flow into the protected block.
    • hopperout will allow items to flow out the protected block.
    • hopper will allow items to flow both directions in the protected block.

  • /cremove

    Will remove all protection on a block. Allowing other players to access and break that block.

  • /cpublic /cprivate /cpassword

    Will set a block to the respective protection if you don't already own that block. (i.e. used /cremove on it)

After inputting one of these commands and is valid, then punch the block to apply this modification to. This will then apply those flags to that particular block.

If you wish to keep a command active, you can use the /cpersist command, which when used as another command is active, you can apply that protection to many items. Inputting the command again disabled it, as it behaves like a toggle.

For more information on the protection system, visit the following page: https://github.com/pop4959/LWCX/wiki/Getting-Started
