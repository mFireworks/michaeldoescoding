Battle Style Randomization Feature

This is a feature I've added to the Universal Pokemon Randomizer FVX Tool that allows you to randomize or modify which style of pokemon battles you wish to use within the generation 3 through 7 games.

Download on Github

This feature originated from the Triple Battles Only Tool I created for Don Shnack, but I always felt it could be made better. Those who used the original tool will know it was a pain to utilize, requiring you to unpackage the game down to it's source files, run the tool, and then repackage it all together. I made the feature for the original branch of this tool, but one of the maintainers of this fork of the project reach out and asked if it could be brought to their active fork. I liked their work and all their active features, and wanted the feature in a released version, so it's been added and is so much easier to use!
