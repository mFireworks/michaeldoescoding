DnD Random Item Generator - Planned Features



  • Levels of Enchantments - Minor, Major, etc
    • 3 Possible levels that could be randomly selected.
    • Each level is different by the amount of possible damage (1D4, 1D6, 1D8).

  • Select subsets of enchantments to select from
    • Allow to omit some enchantments from being in the selection pool.

  • Random "weights" to enchantments
    • Allow for different enchantments to appear more frequently than others.

  • Allow custom enchantments
    • Allows the user to insert their own enchantments into the pools.
    • Save/Load feature to allow easy building of custom tables.
